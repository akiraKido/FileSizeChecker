# FileSizeChecker

## 1. インストール

/Bin の中身をどこかに展開して使ってください。
dllファイルも同じディレクトリに置いてください。

## 2. アンインストール

コピーしたファイルを削除するだけで大丈夫です。

## 3. 使い方

### 3.1 ファイルサイズの一覧を表示
テキストボックスにパスを書くか、[...]ボタンを押してフォルダを開いてください。

### 3.2 ファイルを開く
開きたいディレクトリを右クリックして、「Explorer で開く」を選択してください。

### 3.3 ファイルの詳細を見る
フォルダをダブルクリックすることでその詳細を見ることができます。
